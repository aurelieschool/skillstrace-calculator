Generate Summary Report

  1. Select file from computer from "Browse...".
  2. Select assessment type (please make sure it matches the file)
  3. Click "Upload and Calculate" to process the file.
  4. Once the file is ready for download, click "Download Report" to receive and download the processed report.

If there is an error uploading the file or the downloaded file is blank, please make sure the file matches the selected assessment type and try again. Reselect the file and assessment type as neccessary, then click "upload and calculate" again. Then download the report again.

Also, if you are uploading and several files in a row, make sure to click "upload and calculate" and "download" for every new file.
